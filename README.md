## Usage

`npx @infinumjs/cra-starter [modules] <projectName>`

### Modules

* `--jsonapi`, `-j` - Set up the network stack and [`datx-jsonapi`](https://github.com/infinum/datx#mixins)
* `--storybook`, `-s` - Set up the [storybook](https://storybook.js.org/)
